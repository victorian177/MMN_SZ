◆ はじめに
===========
  日頃から格別のお引き立てを賜りまして厚く御礼申し上げます。

  ここにはCONTEC COMポート番号設定ユーティリティに関する補足説明を記載しています。
  CONTEC COMポート番号設定ユーティリティの使用方法は、オンラインヘルプ(CComUtility.CHM)
  を参照ください。


◆ファイル構成
============================
  ・本製品は、次の様なファイル構成になっております。

  \<User Directory>
  |
  +--\CComUtility
     |
     +--\CComUtility  実行ファイルおよびリソースファイル
     |  |
     |  +--\ja          日本語リソースファイル
     |  |
     |  +--\zh          中国語リソースファイル
     |
     +--\Help         ヘルプファイル
     |  |
     |  +--\JP          日本語版ヘルプファイル
     |  |
     |  +--\US          英語版ヘルプファイル
     |
     +--\Readme       Readmeファイル
        |
        +--\JP          日本語版Readmeファイル
        |
        +--\US          英語版Readmeファイル
